02 SQL 분석 쿼리
SQL 1  —  기상 리스크 × 탄소경제가치
-- 분석기간(25.07~12) 전월 강수량 20mm 이상 → 강수구간 모두 '강우'로 분류됨
SELECT
    CASE
        WHEN t.평균기온 < 5  THEN '한파(5도 미만)'   -- 12월: 1.2°C
        WHEN t.평균기온 < 15 THEN '쌀쌀(5~15도)'     -- 11월: 9.0°C
        WHEN t.평균기온 < 25 THEN '적정(15~25도)'     -- 9월: 23.5°C, 10월: 16.5°C
        WHEN t.평균기온 < 33 THEN '더움(25~33도)'     -- 7월: 28.8°C, 8월: 28.2°C
        ELSE '폭염(33도+)'
    END AS 기온구간,
    CASE
        WHEN r.강수량 = 0   THEN '맑음'
        WHEN r.강수량 <= 10 THEN '소우'
        ELSE '강우'                              -- 실제 데이터 전 구간 해당
    END AS 강수구간,
    SUM(i.이용건수) AS 이용건수,
    ROUND(SUM(i.탄소량), 2) AS 총탄소절감_kg,
    ROUND(SUM(i.탄소량) / 1000 * 9000, 0) AS 탄소경제가치_원
FROM 이용정보 i
JOIN 기온 t ON SUBSTR(i.대여일자, 1, 6) = t.년월
JOIN 강수량 r ON SUBSTR(i.대여일자, 1, 6) = r.년월
GROUP BY 기온구간, 강수구간
ORDER BY 탄소경제가치_원 DESC;
적정(15~25도)·강우 → 탄소경제가치 3,689만원 1위
더움(25~33도)·강우 → 3,338만원 2위
한파(5도 미만)·강우 → 755만원 최저
SQL 2  —  우천 기회비용 추정
-- 추정손실건수 음수 = 평균 대비 초과 이용 발생 월 (7~9월)
SELECT
    r.년월,
    ROUND(r.강수량, 1) AS 월강수량_mm,
    SUM(i.이용건수) AS 실제이용건수,
    ROUND(AVG(SUM(i.이용건수)) OVER(), 0) AS 월평균이용건수,  -- 전체 평균: 약 327만 건
    ROUND(
        AVG(SUM(i.이용건수)) OVER() - SUM(i.이용건수), 0
    ) AS 추정손실건수,                              -- 음수면 초과 이용
    ROUND(
        (AVG(SUM(i.이용건수)) OVER() - SUM(i.이용건수))
        * AVG(i.탄소량) / 1000 * 9000, 0
    ) AS 기회비용_원                               -- 음수면 기회이득
FROM 이용정보 i
JOIN 강수량 r ON SUBSTR(i.대여일자, 1, 6) = r.년월
GROUP BY r.년월, r.강수량
ORDER BY 기회비용_원 DESC;
12월(41mm) → 기회비용 +1억 601만원 최대
11월(21mm) → +1,128만원
9월(371mm) → 초과이용으로 -1억 3,221만원(기회이득)
SQL 3  —  자치구별 탄소 경제효율 · 투자 우선순위
SELECT
    s.자치구,
    COUNT(i.id) AS 총이용건수,
    ROUND(AVG(t.평균기온), 1) AS 평균기온,
    ROUND(AVG(r.강수량), 1) AS 평균강수량,
    ROUND(SUM(i.탄소량), 2) AS 총탄소절감_kg,
    ROUND(SUM(i.탄소량) / 1000 * 9000, 0) AS 총탄소경제가치_원,
    ROUND(SUM(i.탄소량) / SUM(i.이용건수) * 9, 2) AS 건당경제효율_원
FROM 이용정보 i
JOIN 대여소 s ON i.대여소번호 = s.대여소번호
JOIN 기온 t ON SUBSTR(i.대여일자, 1, 6) = t.년월
JOIN 강수량 r ON SUBSTR(i.대여일자, 1, 6) = r.년월
GROUP BY s.자치구
ORDER BY 건당경제효율_원 DESC;

INSIGHT 01
기온이 탄소경제가치를 결정한다
분석기간(25.07~12) 전월 강수량이 20mm 이상으로 강수구간은 모두 '강우'에 해당하며, 기온구간이 탄소경제가치의 핵심 변수로 드러났다. 적정 기온(15~25도)에서 탄소경제가치 약 3,689만원으로 최대, 한파(5도 미만)에서는 755만원으로 5분의 1 수준에 그쳐, 기온 하락이 곧 탄소중립 손실임을 정량화할 수 있다.
→ 기온 리스크 = 탄소경제가치 손실
🌧️
INSIGHT 02
강수량 역설 — 폭우 월이 오히려 이용 급증
강수량이 많은 달의 기회비용이 크다는 가설과 달리, 강수량 최대인 9월(371mm)에 오히려 이용건수가 월평균 대비 71만 건 초과해 기회이득 약 1억 3,221만원이 발생했다. 반면 12월(41mm 소강수)에 기회비용이 1억 601만원으로 최대였다. 기회비용의 실질 원인은 강수량보다 기온(겨울철 이용 기피)임을 시사한다.
→ 기온이 핵심 변수, 우천 가설 재검토 필요
📍
INSIGHT 03
용산·강남·성동 — 탄소효율 상위, 투자 우선순위 1순위
동일 기상 조건에서도 자치구별 건당 탄소효율 격차는 최대 1.89배(용산 6.41원 vs 강서 3.39원). 효율 상위인 용산·강남·성동·동작·서대문구는 이동거리가 길고 출퇴근 목적 이용 비중이 높아 건당 탄소절감량이 크다. 이 5개 자치구에 대여소·거치대를 집중 투자하면 예산 대비 탄소절감 ROI가 최고라는 정책 제언으로 연결된다.
→ 효율 상위 5개 자치구 우선 투자 제언
